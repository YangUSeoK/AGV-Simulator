1. 플래그 생성
원하는 위치에 플래그를 찍는다.

2. Load/Unload 적재장 생성
생성 후, 원하는 플래그에 연결한다.
해당 플래그는 Load/Unload 를 연결하면 해당 속성을 갖는다

적재장 생성 토글 -> 원하는 위치에 클릭으로 적재장 생성 -> 


3. 봇을 생성
Load/Unload 선택 하면, 해당 플래그가 Load/Unload 적재장이 연결 되어있는지 확인한다.
확인이 되면, 해당 플래그를 자신의 Load/Unload 플래그로 설정한다.

4. Play
Load/Unload 플래그에 도착하면, Load Wait/Unload Wait 상태로 전환한다.


Load Wait
현재 적재장에 물건이 있는지 확인한다.
물건이 있으면 Load 상태로 전환한다.

Load
물건을 봇에 싣는다. 
적재장의 카운트를 봇의 적재량 만큼 감소시키고, 봇에 물건을 적재한다.
Move 상태로 전환한다.

Unload Wait
현재 적재장에 물건이 가득차 있는지 확인한다.
빈자리가 있으면 Unload 상태로 전환한다.

Unload
물건을 봇에서 내린다.
적재량 카운트를 증가 시키고, Move 상태로 전환한다.
